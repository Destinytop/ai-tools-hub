# AI Tools Hub

A collection of free AI-powered tools for creators and developers.

## 🚀 Live Demo

[https://ai-tools-hub.vercel.app](https://ai-tools-hub.vercel.app) (即将部署)

## 🛠️ Tools Included

### 1. Sora Prompt Generator
Generate high-quality prompts for Sora AI video generation, optimized for TikTok viral content.

- 16+ professional prompt templates
- TikTok optimization tips
- Style modifiers for cinematic effects
- Variable replacement system

**Keywords**: Sora prompt generator, TikTok video prompts, AI video generation

### 2. DeepSeek-V4 API Cost Calculator
Calculate exact API costs for DeepSeek models including the latest V4.

- Support for all DeepSeek models (V4, V3, Coder-V2, R1)
- Real-time cost estimation
- Token usage presets for common tasks
- Price comparison with GPT-4, Claude, Gemini

**Keywords**: DeepSeek-V4, DeepSeek API cost calculator, AI model pricing

### 3. LLMs.txt Generator
Generate llms.txt files to optimize your website for AI search engines.

- 10 configuration sections
- Real-time preview
- One-click download
- Best practices guide

**Keywords**: llms.txt generator, LLMO, AI search optimization

## 🏗️ Tech Stack

- **Deployment**: Vercel (Static Export)

## 🚀 Deployment

### Vercel (Recommended)

1. Push code to GitHub
2. Import project on [Vercel](https://vercel.com)
3. Deploy automatically

```bash
npm run build
# Output in /dist folder
```
